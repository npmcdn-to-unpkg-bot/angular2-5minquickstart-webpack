## Angular2 turorial(5 Min Quickstart) with Webpack for beginners

This repo is a minimum sample of "Angular2 + Webpack" based on the official 5 Min Quickstart tutorial.

### Install npm modules.
```
npm install
```

### Build JS files.
```
npm run build
```

### Run.
```
npm start
```
